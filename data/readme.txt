Задача: Бинарная классификация. Нужно сделать предсказание того, закажет ли снова клиент конкретный товар.

data - директория с данными для решения задачи предсказания повторной покупки товара пользователем, содержит:

a. train.csv - обучающая выборка:
    row_id - уникальный идентификатор наблюдения
    user_id - уникальный идентификатор пользователя
    product_id - уникальный идентификатор товара
    target - целевая переменная; принимает значение 1, если пользователь повторно купит данный товар в следующем заказе, и 0 в противном случае

b. test.csv - тестовая выборка:
    row_id - уникальный идентификатор наблюдения
    user_id - уникальный идентификатор пользователя
    product_id - уникальный идентификатор товара
    target - целевая переменная; принимает значение 1, если пользователь повторно купит данный товар в следующем заказе, и 0 в противном случае

c. users.csv - таблица с агрегированной информацией о пользователях:
    user_id - уникальный идентификатор пользователя
    total_orders - общее количество предыдущих заказов пользователя
    avg_days_between_orders - среднее количество дней между заказами
    avg_basket_size - среднее количество товаров в заказе
    total_distinct_products - общее количество уникальных товаров, купленных пользователем
    reordered_share - доля товаров, которые пользователь ранее покупал повторно
    favorite_day_of_week - наиболее частый день недели для оформления заказов
    favorite_hour_of_day - наиболее частый час оформления заказов

d. orders.csv - история заказов пользователей:
    order_id - уникальный идентификатор заказа
    user_id - уникальный идентификатор пользователя
    eval_set - тип заказа ("prior","train")
    order_number - порядковый номер заказа пользователя
    order_dow - день недели оформления заказа
    order_hour_of_day - час оформления заказа
    days_since_prior_order - количество дней с момента предыдущего заказа

e. order_items.csv - товары в предыдущих заказах пользователей:
    order_id - уникальный идентификатор заказа
    product_id - уникальный идентификатор товара
    add_to_cart_order - порядковый номер добавления товара в корзину
    reordered - признак того, покупался ли товар пользователем ранее; принимает значение 1, если товар уже заказывался этим пользователем раньше, и 0 в противном случае

f. products.csv - справочник товаров:
    product_id - уникальный идентификатор товара
    product_name - наименование товара
    aisle_id - идентификатор категории второго уровня
    department_id - идентификатор категории верхнего уровня

g. aisles.csv - справочник категорий второго уровня:
    aisle_id - уникальный идентификатор категории второго уровня
    aisle - наименование категории второго уровня

h. departments.csv - справочник категорий верхнего уровня:
    department_id - уникальный идентификатор категории верхнего уровня
    department - наименование категории верхнего уровня

i. data_dictionary.csv - словарь признаков с описанием полей, типов данных и допустимых значений
